# 🚀 CareerForge Pro

### AI-Powered Resume Builder & ATS Analyzer

CareerForge Pro is a full-stack web application that helps users create professional, ATS-friendly resumes using Artificial Intelligence. It combines an intuitive resume builder, live preview, ATS score analysis, AI-powered resume suggestions, and professional PDF generation into a single responsive platform.

Developed as part of a **Web Development Internship at Zaalima Development Pvt. Ltd.**, CareerForge Pro demonstrates the integration of modern frontend technologies, backend APIs, cloud databases, and AI services to simplify resume creation and optimization.

---

# ✨ Features

* 📄 Professional Resume Builder
* 👀 Live Resume Preview
* 🎨 Three Resume Templates
* 🤖 AI Resume Suggestions using Groq API (Llama 3)
* 📊 ATS Score Analysis
* 🔍 Missing Keyword Detection
* 💾 Save Resume to MongoDB Atlas
* ✏️ Update Existing Resume
* 📥 Download Professional PDF Resume
* 📱 Fully Responsive User Interface
* ⚠️ Form Validation & Error Handling

# ⚙️ Installation

### Clone Repository

```bash
git clone <repository-url>
cd CareerForge-Pro
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Runs on:

```
http://localhost:5173
```

### Backend

```bash
cd backend
npm install
npm run dev
```

Runs on:

```
http://localhost:5000
```
